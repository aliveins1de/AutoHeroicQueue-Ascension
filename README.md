# AutoHeroicQueue

WoW-аддон для сервера **Ascension**: автоматически выставляет сохранённый тип
очереди в дропдауне "Тип" окна Dungeon Finder при входе в игру и при каждом
открытии окна, чтобы не выбирать его руками каждый раз.

## Возможности

- Запоминает выбранный тип очереди (например "Random Classic Dungeon Heroic")
  и подставляет его автоматически:
  - через ~3 секунды после входа в игру (`PLAYER_ENTERING_WORLD`);
  - при каждом открытии окна LFD (`OnShow`).
- Команда `/ahq` открывает окно выбора: список всех типов, доступных на
  текущем уровне персонажа, с подсветкой текущего выбора.
- Поддержка скина ElvUI (если он установлен и загружен) с безопасным
  фолбэком на стандартный Blizzard-фон, если модуль скинов недоступен.

## Установка

1. Скачай репозиторий (Code → Download ZIP, либо `git clone`).
2. Положи папку `AutoHeroicQueue` (с файлами `AutoHeroicQueue.toc` и
   `AutoHeroicQueue.lua`) в:
   ```
   <папка игры>/Interface/AddOns/
   ```
3. Перезапусти игру или сделай `/reload`.

## Использование

- `/ahq` — открыть окно выбора подземелья/типа очереди. Выбор сохраняется
  навсегда (через `SavedVariables`) до следующего изменения.
- После выбора аддон сразу попробует применить его к текущему открытому окну
  LFD, если оно открыто.

## Технические детали

Дропдаун "Тип" на Ascension реализован как обычный `UIDropDownMenu`
(`AscensionPVEFrameLFDFrameTypeDropDown`), а не через стандартную
близзардовскую функцию `LFDQueueFrame_SetType`, которой на этом сервере нет.
Поэтому выбор эмулируется программным кликом по нужному пункту в
`DropDownList1` после открытия дропдауна — это работает независимо от
внутренней реализации сервера.

Если после обновления клиента/аддонов Ascension имена фреймов изменятся,
проверь их через `/fstack` (наведя мышь на дропдаун "Тип" и на окно LFD) и
поправь константы в начале `AutoHeroicQueue.lua`:

```lua
local ROOT_FRAME_NAME    = "AscensionLFGFrame"
local LFD_FRAME_NAME     = "AscensionPVEFrameLFDFrame"
local TYPE_DROPDOWN_NAME = "AscensionPVEFrameLFDFrameTypeDropDown"
```

## Отладка

По умолчанию включён debug-вывод в чат (`DEBUG = true` в начале файла).
Чтобы отключить, поставь `DEBUG = false`.
